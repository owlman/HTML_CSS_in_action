# 第 0 章 前言

在当今面向Web前端领域的培训市场上，许多的教程都将他们的主要精力放在了JavaScript语言的使用上，而对于JavaScript语言在前端编程场景中要操作的HTML/XML文档、CSS样式表等计算机标记语言及其相关框架的使用方法则往往介绍得过于简单，甚至只是蜻蜓点水，一带而过。以这种方式培养出来的前端工程师在实际工作中往往只能以JavaScript编程的视角来理解前端项目所需要完成的任务，即如何调用DOM/BOM等对象模型提供的API来向Web应用程序的后端发送请求，并解析其响应给前端的数据，而对于如何使用HTML/XML合理安排用户界面的结构，如何使用CSS来设置用户界面中所要呈现的界面元素，则往往知其然而不知其所以然，最后不得不求助一些工作思维完全不同的美工人员或者购买他们提供的网页设计模版，而这又会给项目团队带来更多的额外开销。

为了解决上述问题，卷积传媒重新开发了一套面向前端程序员的培训课程。本书作为该系列课程中的第一本配套教材，将首先为初学者介绍如何在Web前端开发项目中使用HTML+CSS这两门计算机标记语言及其相关框架来完成网页设计部分的任务。待读者掌握了这些基础技能之后，后续教材才会再进一步介绍如何使用JavaScript语言来完成Web应用程序前后端的编程技能。

## 0.1 本书简介

就具体内容而言，本书除前言部分外共包含5章内容，每一章都对应着一个网页设计项目，每个项目都对应着一个企业网站开发过程中的一个具体需求，笔者将基于这些项目的实现过程来为大家介绍如何使用HTML+CSS这两门计算机标记语言及其相关框架来完成网页设计部分的任务。下面来具体罗列一下这些项目：

- **项目1：企业的线上名片设计**：在这一章中，笔者将会带领读者创建一个用于推广新创企业的“线上名片”项目。该演示项目的目标是让读者初步了解网页设计工作所需要搭建的软件环境，以及实施此类项目所要执行的基本步骤，以及执行这些步骤所需掌握的基本知识。
- **项目2：企业网站的首页设计**：在这一章中，笔者将会以一家名为“凌雪冰熊”的连锁饮料店的需求为例来演示如何完成企业网站的首页设计。该演示项目的设计目标是向公众展示该饮料店的品牌信息，以便建立起人们对这家连锁品牌的第一印象。同时，该项目也会为该企业网站后续要设计的新闻活动、产品展示、申请加盟等页面设置好统一的布局风格与配色方案，并预留整合这些页面的导航链接。通过本章项目的实践，读者将会初步了解设计一个企业展示类页面所要执行的基本步骤，以及执行这些步骤所需的基本技术与相关工具。
- **项目3：企业网站的新闻活动页设计**：在这一章中，笔者会继续以凌雪冰熊连锁饮料店的需求为例来演示如何为企业网站设计“新闻活动”页，该演示项目的设计目标是向公众展示企业的最近经营活动以及获得的业绩和社会影响力，以便加强人们对这家连锁饮料店的了解与信心。同时，该网页的设计也必须要延续该网站首页建立起来的布局风格与配色方案，并同样在导航栏中预留跳转到网站首页、产品展示、申请加盟等页面的链接。通过本章项目的实践，读者将会初步了解完成一个图文信息类页面的排版任务所要执行的基本步骤，以及执行这些步骤所需的基本技术与相关工具。
- **项目4：企业网站的产品展示页设计**：在这一章中，笔者会继续以凌雪冰熊网站中“产品展示”页的设计为例来演示如何为企业网站设计出类似于哔哩哔哩的多媒体播放界面。该项目的设计目标是为凌雪冰熊网站提供视频、动画等多媒体资料的播放页面，以便更直观地展示这家连锁店最近的主打产品与新加盟的门店，从而吸引到更多潜在的合作伙伴。同样的，该网页在外观设计上也必须要延续该网站首页建立起来的布局风格与配色方案，并同样在导航栏中提供跳转到网站首页、新闻活动、申请加盟等页面的链接。通过本章项目的实践，读者将会初步了解设计一个多媒体播放类页面所要执行的基本步骤，以及执行这些步骤所需的基本技术与相关工具。
- **项目5：企业网站的申请表单页设计**：在这一章中，笔者会继续以凌雪冰熊网站中“申请加盟”页的设计为例来演示如何为企业网站设计出用于提交数据的表单页。该演示项目的设计目标是为潜在的客户提供提交加盟申请的用户界面，以便简化凌雪冰熊这家连锁饮料店的加盟流程，增加潜在客户加盟的意愿。同样的，该网页的设计也必须要延续该网站首页建立起来的布局风格与配色方案，并同样在导航栏中预留跳转到网站首页、新闻活动、产品展示等页面的链接。通过本章项目的实践，读者将会初步了解设计一个用于输入性的用户界面所要执行的基本步骤，以及执行这些步骤所需的基本技术与相关工具。

## 0.2 读者须知

由于这是一本专注于介绍如何基于HTML+CSS这两门计算机标记语言，以及Bootstrap这个网页设计框架来完成网页设计任务的教材，因此，笔者在本书中会尽可能地避免使用JavaScript、Python等编程语言。如果读者想要了解与Web应用程序前后端编程相关的这些技术。可以在阅读完本书，并确认掌握了HTML+CSS及其相关的基础知识后，再通过继续阅读本系列课程的后续配套教材来完成更进一步的学习任务。

当然了，由于网页设计领域的框架不仅琳琅满目，选择众多，而且新陈代谢极为快速，这意味着等到这本书写完并最终出版之时，读者在Bootstrap框架之外很有可能已经有了更好的选择。所以基于“授之以鱼不如授之以渔”的原则，本书的真正目的是希望帮助读者掌握快速学习任意一种网页设计框架的能力，该能力才是程序员们在这个快速发展的时代可持续发展的源动力所在。

除此之外，笔者在这里还需要特别强调一件事：本书中所有示例项目的实现都是基于各章节中的代码占比及其阅读体验等众多写作因素进行了平衡考虑之后产生的最简化版本，其中省略了相当数量的、属于细节性质的代码。因此，如果想了解这些项目中某些细节问题的解决方案，还需请读者自己去查阅本书附带的源码包。当然了，在笔者个人看来，如果想要学好并熟练掌握一项计算机技术，最好的办法就是尽可能地在实践中使用它们，在实际项目需求的驱动下模仿、试错并总结使用经验。所以在这里，本书并不鼓励直接复制/粘贴的本书附带源码包中的代码，我们更期待读者“自己动手”去模仿书中提供的项目，亲手将自己想要执行的代码输入到计算机中，观察它们是如何工作的。然后，试着修改它们，并验证其结果是否符合预期。如果符合预期，就总结当下的经验，如果不符合预期，则去思考应该做哪些调整来令其符合预期。如此周而复始，才能让学习效果事半功倍。

最后，感谢正在阅读的您选择了这本书，希望它能够帮助您更好地掌握使用HTML+CSS这两门计算机标记语言的知识和技术，并在实际应用中得到提升和应用。我们也欢迎读者提出宝贵的意见和建议，以帮助我们不断改进和完善本书。
